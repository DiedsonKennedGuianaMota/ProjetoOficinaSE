O que são estruturas condicionais e por que elas são usadas.
Operadores relacionais: >, <, >=, <=, == e !=.
Operadores lógicos: &&, || e ! para combinar condições.
Exemplos em C com if, else e else if.

Exemplo rápido em C

#include <stdio.h>

int main() {
    int idade = 16;

    if (idade >= 18) {
        printf("Maior de idade\n");
    } else {
        printf("Menor de idade\n");
    }

    return 0;
}
